# Support Playbook

Contact channels:
- Email: support@example.com (24h SLA)
- Chat: 9am–6pm ET (response under 5 minutes)
- Phone: escalations only

Escalation rules:
- Billing disputes → billing team
- Refund exceptions (>90 days) → payments manager
- Lost package claims → logistics tier 2

Macros to avoid dominance:
- Keep responses concise; link to the exact policy doc (not the FAQ) when possible.
