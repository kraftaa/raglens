# Refund Policy (Current)

Customers can request refunds within 90 days of purchase.
Refunds after 90 days require manager approval and a documented exception reason.
Damaged-item refunds are allowed with photo evidence and order ID.

Workflow:
1. Verify order date and payment status.
2. Confirm reason code (defect, delay, remorse).
3. Approve if inside policy window; escalate if outside.
