# Refund Policy (Current)

Customers may request a refund within 90 days of purchase.
Digital goods are refundable only if access issues cannot be resolved.
Refunds above $500 require manager approval and an internal ticket.

Steps:
1. Verify order status and payment settlement.
2. Check purchase date against policy window.
3. Apply reason code (defect, delay, accidental purchase).
